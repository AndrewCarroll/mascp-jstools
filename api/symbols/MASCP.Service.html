Class MASCP.Service

Super-class for all MASCP services to retrieve data from proteomic databases. Sub-classes of this class override methods to change how requests are built, and how the data is parsed.
Defined in: MascpService.js.

retrieve()
Asynchronously retrieves data from the remote source. When data is received, a resultReceived.mascp event is triggered upon this service, while an error.mascp event is triggered when an error occurs. This method returns a reference to self so it can be chained.
