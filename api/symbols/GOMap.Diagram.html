Method Detail
<static> {Element} GOMap.Diagram.addZoomControls(zoomElement, min, max, precision, value)
Given an element that implements a zoom attribute, creates a div that contains controls for controlling the zoom attribute. The zoomElement must have a zoom attribute, and can fire the zoomChange event whenever the zoom value is changed on the object. The scrollwheel is connected to this element so that when the mouse hovers over the controls, it can control the zoom using only the scroll wheel.
Parameters:
{Object} zoomElement
Element to control the zooming for.
{Number} min
Minimum value for the zoom attribute (default 0)
{Number} max
Maximum value for the zoom attribute (default 10)
{Number} precision
Step precision for the zoom control (default 0.5)
{Number} value
Default value for this control
Returns:
DIV element containing the controls
